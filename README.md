# Resume Skill Extractor (Modern Web App)

## Backend (FastAPI)
- `cd backend`
- `pip install -r requirements.txt`
- `python -m spacy download en_core_web_sm`
- `uvicorn main:app --reload`

## Frontend (React)
- `cd frontend`
- `npm install`
- `npm start`

Visit [http://localhost:3000](http://localhost:3000) in your browser!
